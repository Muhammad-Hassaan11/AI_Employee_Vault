# Instagram + Facebook Content Calendar — AI Agents / Automation

7-day content sprint (2026-07-12 → 2026-07-18), 3 posts/day, each post
cross-posted to both Instagram and Facebook from the same creative. All
drafts live in /Pending_Approval as `instagram_post` / `facebook_post`
files — nothing publishes until the human flips `status: pending` to
`approved` (see [[Company_Handbook]]).

Only these image aspect ratios are used, per the human's instruction:
**Square 1:1**, **Portrait 4:5**, **Landscape 1.91:1**. No image files
were generated (no image-gen/hosting tool is available to Atlas) — each
post has a detailed **Image Brief** instead. Before approving an
`instagram_post` file, fill in its `image_url:` field with a real public
URL for the image once it's produced from the brief (Canva/Midjourney/
designer). `facebook_post` files don't require a URL in frontmatter, but
should carry the matching image when posted manually if the platform
needs it attached.

Daily rhythm: Slot A (~9:00 AM) = Square 1:1 · Slot B (~1:00 PM) =
Portrait 4:5 · Slot C (~6:00 PM) = Landscape 1.91:1.

## Posting Guidelines

- Voice: professional, practical, no hype (matches [[Content_Calendar|LinkedIn Content Calendar]])
- Every post ends with a soft CTA (DM us / comment / book a call)
- Instagram: 3-5 relevant hashtags. Facebook: plain text, no hashtag clutter
- Same creative (one image brief per slot) is used for both platforms —
  only the caption is platform-tuned
- All 42 drafts (21 topics x 2 platforms) start at `status: pending` in
  /Pending_Approval; nothing publishes without the human approving
