# Skills Index (Map of Content)

Atlas's Agent Skills — step-by-step playbooks living in `/Skills/*/SKILL.md`.
Every skill file is named `SKILL.md` (the Agent Skills convention), so they
all appear as **"SKILL"** in the graph; hover or click a dot to see which one.

- [[process-task skill]] — process a new /Needs_Action task end to end
- [[summarize-file skill]] — condense a file into a summary + action items
- [[send-email skill]] — draft an email and route it through approval
- [[linkedin-post skill]] — draft a sales LinkedIn post for approval
- [[social-post skill]] — draft Facebook / Instagram / X posts + summaries
- [[handle-approvals skill]] — work the /Pending_Approval queue
- [[weekly-audit skill]] — run the weekly audit → CEO briefing
